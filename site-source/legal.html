隐私政策与使用条款

Education Advisor 坚持本地优先与隐私保护。以下条款说明了数据处理方式与软件使用授权。

隐私政策

最后更新日期：2026 年 7 月 16 日

1. 数据存储方式

Education Advisor 是一款本地优先的桌面应用。您的学生信息、事件记录、学业数据、配置文件等均存储在您的本地设备中，默认不会上传到任何云端服务器。

2. 外部 API 调用

当您配置并使用 AI 智能体时，应用会根据您的设置向第三方 LLM 提供商（如 OpenAI、Anthropic、DeepSeek 等）发送请求。在此过程中：

  • PII 隐私引擎会优先对敏感信息进行脱敏处理；
  • API 请求遵循您选择的提供商的隐私政策；
  • 我们不会拦截、存储或分析您的对话内容。

3. 数据安全

应用使用本地 SQLite 与 JSONL 文件存储数据，敏感映射表采用 AES 加密。建议您妥善保管设备与备份文件。

4. 儿童隐私

Education Advisor 面向教育工作者设计，不直接面向未成年人收集个人信息。使用本软件处理学生数据时，请遵守当地教育数据保护法规。
