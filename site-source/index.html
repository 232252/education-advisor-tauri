核心特性

把班主任的重复劳动
交给 AI 智能体

从德育评分到家校沟通，从学业预警到班级日志，Education Advisor 用多智能体协作把繁杂事务变得井井有条。

操行评分与德育管理

加扣分一键录入，自动汇总周/月操行排名。支持自定义评分维度与权重，德育分、纪律分、卫生分分类管理，班级表现一目了然。

学业成绩分析

导入月考/期中/期末成绩，自动生成班级排名、学科均分、进步退退榜。AI 智能识别成绩波动学生，生成个性化学习建议与学业报告。

风险预警与关怀

综合操行、学业、出勤多维度数据，AI 自动识别需重点关注的学生。风险等级分级提醒，生成关怀建议，帮助教师及时干预。

家校沟通与通知

AI 自动生成个性化家校沟通稿，根据学生表现智能匹配沟通策略。支持群发通知、个别沟通记录追踪，家校协作更高效。

班级日志与周报

日常事件流水记录，自动生成班级日志和周报。AI 智能体按周总结班级动态、突出问题和改进建议，让班级管理有据可查。

Rust 高性能引擎

EAA CLI v3.2.3 数据引擎，流式处理 + 多级缓存。score 2ms、ranking 6ms、stats 3ms，数万次操作零错误，毫秒级响应。

本地优先，隐私保护

数据保存在本地，PII 隐私引擎支持脱敏、反脱敏与按接收者过滤，敏感信息不上云。AES 加密映射表保障数据安全。

18 个专业 AI 智能体

辅导员、监督员、学业指导、风险预警、家校沟通、数据分析师等，各司其职。支持 35+ LLM 提供商，按需编排协作。

MCP 与 19 个 AI 工具

支持 Model Context Protocol，11 个 EAA 能力门控工具 + 6 个文件工具 + 2 个通用工具，AI 可安全访问全部数据。

常见问题

有疑问？这里可能有答案

Education Advisor 是免费的吗？

是的。Education Advisor 采用 MIT 协议开源，完全免费使用。项目源码托管在 GitHub，欢迎提交 Issue 或 PR。

数据会上传到云端吗？

不会。Education Advisor 采用本地优先架构，学生信息、事件记录、学业数据等均保存在本地设备。只有在调用外部 LLM API 时才会按需发送请求，且 PII 隐私引擎会先对敏感信息进行脱敏处理。

支持 macOS 或 Linux 吗？

当前 v0.1.0 仅发布 Windows x64 安装包。由于项目已迁移至 Tauri 2，跨平台构建已具备技术基础，macOS 与 Linux 版本将在后续版本中提供。

为什么安装包这么小？

应用核心 UI 由 Tauri 2（Rust）提供，仅约 3–5 MB；业务逻辑通过 Node.js Sidecar 运行，首次运行时会检查本地 Node.js 环境。相比传统 Electron 打包，体积大幅减小。

可以自定义 AI 提供商吗？

可以。Education Advisor 支持 OpenAI、Anthropic、DeepSeek、Moonshot、Google、Ollama 等 35+ 个 LLM 提供商，并支持在设置中配置自己的 API Key 与模型参数。
